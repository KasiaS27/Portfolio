# 🎨 UI/UX Portfolio

Strona prezentuje moje projekty i case studies.

Projekt jest w **ciągłej rozbudowie** – regularnie dodaję nowe projekty i ulepszam istniejące.

🌐 **Live:** https://kasiaskrobiszportfolio.netlify.app



# 📖 O projekcie

To moje osobiste portfolio, na którym prezentuję projekty z zakresu UX/UI designu. Każdy projekt ma swoją stronę z case study, gdzie opisuję proces pracy – od researchu, przez projektowanie, aż po testowanie z użytkownikami.

Strona zbudowana jest w **HTML5**, **CSS3** i **JavaScript**. Portfolio zostało zaprojektowane i zaimplementowane jako własny projekt prezentujący umiejętności z zakresu UX/UI oraz front-end developmentu.

Strona zawiera case studies dla projektów takich jak:
- Research dla sektora travel
- Admin panel
- Interfejs dla medycyny
- i inne...


# ✨ Zakres projektu

- przygotowanie struktury strony (HTML5),
- stylowanie i responsywny design (CSS3),
- interaktywne elementy i animacje (JavaScript),
- integracja bibliotek (Bootstrap, Swiper, GLightbox, Typed.js),
- obsługa formularza kontaktowego (Web3Forms),
- case studies dla poszczególnych projektów,
- responsive design dla komputerów, tabletów i mobilów,
- wdrożenie na Netlify.



# 🛠️ Technologie

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5.3.2
- Bootstrap Icons
- Swiper
- GLightbox
- Typed.js
- Web3Forms
- Netlify



# 📚 Czego nauczyłam się podczas realizacji?

Projekt pozwolił mi lepiej zrozumieć proces tworzenia portfolio oraz praktycznie zastosować umiejętności z zakresu frontend developmentu.

Podczas pracy rozwinęłam umiejętności:

- strukturyzacji projektu HTML,
- zaawansowanego stylowania CSS,
- pisania czystego kodu JavaScript,
- integracji gotowych bibliotek,
- responsywnego designu,
- deploymentu i hostingu.

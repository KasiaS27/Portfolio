# UI/UX Portfolio Katarzyna Skrobisz


W tym repozytorium znajduje się kod źródłowy mojego portfolio oraz wszystkie zasoby projektu.

🌐 **Zobacz działającą stronę:** https://kasiaskrobiszportfolio.netlify.app

## 💁‍♀️ O projekcie

To moje osobiste portfolio prezentujące wybrane projekty, nad którymi pracowałam. 
Portfolio zostało stworzone jako przykład mojej pracy z HTML, CSS i JavaScript oraz jako wizytówka moich umiejętności UX/UI.

Strona zawiera:
- sekcję **O mnie** z krótką prezentacją mojej osoby i doświadczenia,
- portfolio projektów z miniaturami i opisami,
- dane kontaktowe.

Portfolio zostało zaprojektowane z naciskiem na czytelność oraz intuicyjność.

## 🛠 Technologie

Projekt został stworzony przy użyciu:

- HTML5
- CSS3
- JavaScript
- statycznego hostingu na Netlify
